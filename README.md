## 🎯 Proyectos Destacados

### 1. **FinTrack – App de Finanzas con IA**
**Descripción**: Dashboard moderno para gestión integral de finanzas personales con inteligencia artificial integrada.

**Características principales**:
- 📊 **Visualización de datos avanzada** con ApexCharts
- 💬 **Chatbot inteligente** basado en API de Gemini
- 💰 **Análisis de gastos automático** con IA
- 📈 **Dashboard responsive** optimizado para todos los dispositivos
- 🔐 **Autenticación segura** de usuarios

**Stack**:
- Frontend: React, ApexCharts
- Backend: Node.js, Express
- BD: MySQL
- IA: Google Gemini API

**Enlace**: [Ver en GitHub](https://github.com/franPdev/FinTrack)

### 2. **WeatherNow – Clima en Tiempo Real**
**Descripción**: Aplicación web que proporciona información meteorológica en tiempo real con interfaz visual dinámica.

**Características principales**:
- 🌤️ **Datos de clima en tiempo real** via OpenWeather API
- 🎨 **Fondos dinámicos** que cambian según el clima
- 📱 **Interfaz mobile-first** totalmente responsive
- 🔍 **Búsqueda de ciudades** intuitiva y rápida
- 🌐 **Geolocalización automática** del usuario

**Stack**:
- Frontend: React, CSS3 animations
- API: OpenWeather API
- Hosting: Optimizado para dispositivos móviles

**Enlace**: [Ver en GitHub](https://github.com/franPdev/weathernow)
